# Getting Started

# Running Project

## Navigate to Project

```bash
cd /path/to/project
```

## Open IDE
This command will open vs code.
```bash
code .
```

## Install Dependencies

```bash
# using npm
npm install
```

```bash
# OR using Yarn
yarn install
```

```bash
# OR using Expo
npx expo install
```

## Start Packager

```bash
# using npm
npm start
```

```bash
# OR using Yarn
yarn start
```

## Run on device
Iphone
```bash
# Press i
i
```
Android
```bash
# Press a
a
```
Web
```bash
# Press w
w
```
This wil install Expo go in native device and app will run.
